# AI Course Projects - FEUP 2025

Repository containing projects developed for the Artificial Intelligence course at FEUP (Faculty of Engineering of the University of Porto) during the 3rd year, 2nd semester (2025).

## Project Structure

### Project 1: Game AI Bot
Located in `Proj1/` - Implementation of an AI bot for a game using search algorithms and heuristics.

### Project 2: Supervised Learning
Located in `Proj2/` - Machine learning project implementing supervised learning algorithms.

## Technologies Used

- **Python 3.10+**
- **Pygame** (Project 1)
- **NumPy & Pandas** (Project 2)
- **Scikit-learn** (Project 2)
- **Jupyter Notebook** (Project 2)
